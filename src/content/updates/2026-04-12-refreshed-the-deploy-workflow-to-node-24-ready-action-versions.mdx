The deploy workflow was already succeeding, but GitHub still warned that several marketplace actions were running on deprecated Node 20 action runtimes. The workflow now uses the current major releases for checkout, setup-node, pnpm setup, and AWS credential configuration so the publish path stays compatible with the GitHub-hosted runner changes scheduled for 2026.

This is a maintenance update rather than a behavior change. The deploy contract stays the same: build the Astro site, assume the tracked GitHub OIDC role, sync the static output to S3, and invalidate the CloudFront distribution.

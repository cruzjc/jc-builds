GitHub's runner deprecation note now gives a concrete date for the default runtime shift: JavaScript-based actions begin moving to Node 24 on June 2, 2026. The deploy workflow already used current action majors, but `aws-actions/configure-aws-credentials@v5` still showed one remaining Node 20 warning in the run annotations.

This update opts the deploy job into `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24=true` now instead of waiting for the platform default to change. That keeps the S3 and CloudFront publish path on the future runner behavior immediately, gives earlier signal if any marketplace action still breaks under Node 24, and removes the last deprecation warning from the deployment pipeline.

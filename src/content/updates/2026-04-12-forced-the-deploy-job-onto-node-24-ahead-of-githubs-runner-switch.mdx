GitHub's runner deprecation note now gives a concrete date for the default runtime shift: JavaScript-based actions begin moving to Node 24 on June 2, 2026. The deploy workflow already used current action majors, but `aws-actions/configure-aws-credentials@v5` still publishes a Node 20 target in its action metadata.

This update opts the deploy job into `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24=true` now instead of waiting for the platform default to change. The latest deployment run confirms the job is using the Node 24 path already, even though GitHub still emits an annotation because the AWS credentials action itself has not been republished with a Node 24 target yet.

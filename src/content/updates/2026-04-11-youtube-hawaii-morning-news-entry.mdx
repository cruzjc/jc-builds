This entry brings a second real project into the public registry instead of leaving it as a private local repo.

The published record makes the operating model explicit: Honolulu-time scheduling, topic-based playlist creation, encrypted YouTube OAuth storage, and retention-driven cleanup for app-managed playlists.

It also captures an important infrastructure boundary for future work. This app should keep its own Google Cloud project for OAuth and the YouTube Data API so credentials, quota, and redirect configuration stay scoped to the project instead of being shared across unrelated tools.

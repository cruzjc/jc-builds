AI News Hub is now live as a public AWS-hosted surface instead of only existing as a local scaffold.

The deployed footprint includes a CloudFront site, an HTTP API, Lambda runtimes for the API, MCP, and ingest pipeline, a DynamoDB content table, an SQS enrichment queue, a Step Functions state machine, and an hourly EventBridge trigger.

This first production pass is intentionally narrow. The public experience is live and the infrastructure is real, but the surfaced content is still the seed-backed contract while the ingest pipeline and persistence model mature into the next iteration.

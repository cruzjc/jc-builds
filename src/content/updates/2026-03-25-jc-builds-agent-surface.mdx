The design question was whether semantic HTML would be enough for visiting agents.

It helps, but it is not sufficient on its own when a system wants a fast summary of the whole site. The compromise is straightforward:

- keep the HTML rich and readable
- also publish explicit JSON indexes
- add `llms.txt` as the shortest route into the content map

That keeps the human experience intact while removing friction for automation.

## Objective

Seamless QR turns a picture into a quiet doorway. It is designed for invitations,
posters, personal artwork, and small-business promotions where a conventional
black-and-white code would interrupt the composition.

## How it works

Classic turns the former Safe, Balanced, and Stealth presets into one continuous
blend-strength control, with manual position, size, and rotation controls plus
a placement preview. Ghost Lab expands the local search across QR correction,
masks, nearby angles, and scales while ranking texture, edge alignment,
saliency, finder contrast, and required tonal change.

Generative Lab is a separate, explicit opt-in path. A deterministic
perceptual/EOT optimizer selects texture-aware placement, size, and rotation,
then runs as a short-lived one-L4 Cloud Run Job. It is not a diffusion model and
does not regenerate the source artwork. Cloud-side OpenCV and ZXing-C++ checks
are followed by exact-payload jsQR and ZXing JS validation in the browser before
download.

## Current state

The public beta supports anonymous static HTTP and HTTPS destinations through
Classic and the screen-first Ghost Lab experiment. Ghost still requires exact
pristine decoding in both browser engines; informed by successful real-phone
testing, its stress gate requires six of twelve variants independently in each
decoder. It can return Most hidden, Best balance, and Most reliable validated
poses without rerunning.

The renderer treats the source artwork and QR-driven correction as separate
working layers, but export deliberately flattens them into one metadata-stripped
PNG. Classic and Ghost remain local, so pictures never leave the device.
Generative Lab is temporarily protected by a PIN while the end-user flow is
tested. It uploads only after explicit consent, deletes source artwork at
terminal job cleanup, and retains a one-day storage lifecycle as a fallback
rather than a guaranteed deletion time.

The cloud path allows one active GPU execution, supports user cancellation, and
closes new work at $40 of tracked monthly spend, cancels an active execution at
$45, and emits a project-scoped $50 budget alert. The alert is not a hard billing
cap. A final production-path smoke passed both cloud decoder gates, while the
browser validators decoded the exact normalized destination in 12 of 12
deterministic variants in jsQR and 12 of 12 in ZXing JS. It left no source,
queue task, lock, or dispatch record behind.

The product does not promise universal invisibility or a commercial scan
guarantee. Users should test final artwork on the intended phone, display, and
print size. Payment, accounts, saved projects, editable links, and diffusion or
regenerative image generation remain outside this release.

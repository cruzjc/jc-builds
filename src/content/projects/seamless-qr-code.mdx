## Objective

Seamless QR turns a picture into a quiet doorway. It is designed for invitations,
posters, personal artwork, and small-business promotions where a conventional
black-and-white code would interrupt the composition.

## How it works

The creator runs entirely in the browser. It tests Q and H error correction
across every QR mask while searching rotation, placement, scale, and six signal
levels. The adaptive renderer only moves pixels that need more local contrast,
leaving already camera-readable artwork untouched and retaining source color.
Every downloadable result must be discovered in the full artwork, decode to the
exact link in both jsQR and ZXing, and meet the selected per-decoder stress-test
threshold.

## Current state

The public beta supports anonymous static HTTP and HTTPS destinations, Safe,
Balanced, and experimental Stealth scan targets, automatic or manual placement,
separate scan and tonal-change reporting, reproducible full-resolution export,
and PNG download. Pictures never leave the device. Accounts, editable Event
Links, billing, saved projects, analytics, and QR-conditioned GPU art remain
deliberate follow-on work.

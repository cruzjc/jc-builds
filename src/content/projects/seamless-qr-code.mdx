## Objective

Seamless QR turns a picture into a quiet doorway. It is designed for invitations,
posters, personal artwork, and small-business promotions where a conventional
black-and-white code would interrupt the composition.

## How it works

The creator runs entirely in the browser. Classic turns the former Safe,
Balanced, and Stealth presets into one continuous blend-strength control, with
manual position, size, and rotation controls plus a placement preview. Ghost
Lab expands the search to M/Q/H correction, all QR masks, nearby geometric
angles around the user's preferred rotation, and scales from 22% to 88% while
ranking local texture, edge alignment, saliency, finder contrast, and required
tonal change. Every
downloadable result must be discovered in the full artwork, decode to the exact
link in both jsQR and ZXing, and meet the selected per-decoder stress threshold.

## Current state

The public beta supports anonymous static HTTP and HTTPS destinations through
Classic and the screen-first Ghost Lab experiment. Ghost still requires exact
pristine decoding in both engines; informed by successful real-phone testing,
its stress gate now requires six of twelve variants independently in each
decoder. It can return Most hidden, Best balance, and Most reliable validated
poses without rerunning.

The renderer treats the source artwork and QR-driven correction as separate
working layers, but export deliberately flattens them into one metadata-stripped
PNG. Classic and Ghost remain local, so pictures never leave the device. An
opt-in Generative Lab using semantic placement and QR-conditioned GPU inpainting
has been researched, but no cloud image upload, GPU service, storage, or queue
has been provisioned. Accounts, editable Event Links, billing, saved projects,
and analytics also remain follow-on work.
